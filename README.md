This interactive HTML-based safety system provides quick access to:

- **MCC Panels** - 8 Motor Control Center panels with 270+ circuits
- **Task LOTO Procedures** - 54 task-based lockout/tagout procedures
- **Equipment LOTO Procedures** - 20 equipment-based lockout/tagout procedures
- **Hazard Assessments (HA)** - Workplace hazard analysis documents
- **Job Safety Analysis (JSA)** - Step-by-step job hazard breakdowns
- **Job Assessments (JA)** - Job assessment documents
- **Job Summaries** - Position descriptions
- **SDS/MSDS** - Safety Data Sheets
- **Policy's/Manuals** - Company policies and manuals
- **Inspections** - Inspection documents and checklists
